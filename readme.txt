== BREW GL demo ==
Keypad:
 1 	- Enable Blending
 2 	- Diable Blending
 4 	- Enable Lighting
 5 	- Disable Lighting
 7 	- Enable Texturing
 8 	- Disable Texturing
 * 	- Enable Fog
 0 	- Disable Fog
 UP 	- Zoom out
 DOWN 	- Zoom in
 LEFT 	- Move cube left
 RIGHT 	- Move cube right
 OK 	- Reset cube position
